# NPB/KBO 總分模擬分析系統 🎌🇰🇷

這是一個使用 Python 撰寫的日職（NPB）與韓職（KBO）比賽總分模擬分析工具，透過蒙地卡羅、負二項、泊松分布三種統計模型，預測比賽總分是否超過盤口。

## 🔧 功能說明

- 完整模型（含投手與團隊打擊資料）
- 基準模型（僅使用基本得分資料）
- 三種機率模擬：蒙地卡羅、負二項、泊松分布
- 提供推薦指數與建議方向

## ▶️ 使用方式

1. 安裝所需套件：

```
pip install -r requirements.txt
```

2. 執行分析程式：

```
python NPBKBO.py
```

3. 輸入相關比賽資訊與球隊資料後，即可看到模型分析與推薦。
